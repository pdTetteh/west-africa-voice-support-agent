# Contributing

Thank you for your interest in contributing to West Africa Voice Support Agent.

This project is an open-source prototype of a voice-first support copilot for low-resource West African customer-service settings. It combines local ASR, retrieval-grounded support responses, ticketing, knowledge-base management, and evaluation.

## Good contribution areas

Useful contributions include:

- improving ASR handling for noisy and accented speech
- adding multilingual or code-switched examples
- improving retrieval and ranking
- improving grounded answer generation
- expanding evaluation datasets and metrics
- improving the frontend dashboard
- adding tests
- improving documentation
- improving deployment support
